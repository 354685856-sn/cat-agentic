# x-agentic-workflow v0.16.0

This release tightens desktop session history behavior by making session times
stable and explicit.

## Highlights

- **Stable session time labels**: desktop session summaries now include
  month-day/time labels instead of browser-relative time strings.
- **Persisted-time ordering**: the desktop session list is ordered by the saved
  session update timestamp, not only by session id.
- **History-open guard**: opening an existing session does not refresh its
  `updated_at` value.
- **Clean-room reference discipline**: this follows public release-note lessons
  around history timestamps and list stability without using external
  implementation details.
